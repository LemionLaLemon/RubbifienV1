Rubbifien is a messaging app similar to [Slack](https://slack.com/) or [Discord](https://discord.com/) built on plain HTML JS and CSS using firebase to store and deliver messages.

## Features:
- You can *enter in messages*
- You can *send the messages*
- You can *view the messages*
- You can *make an account*
- You can *change your username*
> [!Tip]
> you can now view messages in a whole **REAL TIME**!!!
